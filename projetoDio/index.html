Tags

(Voltar)

No HTML, as Tags sao usadas para estruturar e definir o conteúdo de uma página. Elas são representadas por meio de marcações angulares e geralmente são compostas por um nome que indica a função do elemento. Existem muitas tags disponíveis no HTML, cada uma com uma finalidade específica.

HTML : Define o início e o fim do documento HTML.

Head : Contém informações sobre o documento, como o título da página e referências a arquivos externos.

Body : Engloba o conteúdo principal da página, como texto, imagens, links, etc.

H1 até H6 : São as tags para cabeçalhos, onde H1 é o cabeçalho mais importante e o H6 é o menos importante.

P : Utilizada para criar parágrafos de texto.

A : Cria um link para outra página ou recurso externo.

IMG : Insira uma imagem na página.

UL e LI : Usadas para criar listas não ordenadas, onde UL define uma lista e LI define cada item da lista.

Links

(Voltar)

Usar links no HTML, significa criar elementos interativos na página da web, permitindo que os usuários naveguem para outras páginas ou acessem recursos externos ao clicar em um texto, imagem ou área específica da página.

Os links são criados por meio da tag 'a' (âncora) e são definidos pelo atributo href , que especifica o destino do link. O conteúdo do link, que pode ser texto ou uma imagem, é colocado entre as tags de abertura e fechamento '/a' .

Listas

(Voltar)

No HTML, você pode criar listas ordenadas e não ordenadas usando as tags 'ul' (lista não ordenada) e 'ol' (lista ordenada), juntamente com uma tag 'li' (item da lista) para cada item da lista.

Os itens da lista são colocados dentro das tags 'li'. A tag 'ul' é usada para envolver todos os itens da lista não ordenada. Isso será exibido no navegador como:

  • Item 1
  • Item 2
  • Item 3

Assim como na lista não ordenada, os itens são colocados dentro das tags 'li'. No entanto, nesse caso, uma tag 'ol' é usada para envolver todos os itens da lista ordenada.

  1. Item 1
  2. Item 2
  3. Item 3
